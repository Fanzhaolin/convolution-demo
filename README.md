# 连续信号卷积运算智能体

## 📋 项目介绍
这是一个实时可视化卷积运算的网页应用，支持自定义函数表达式和交互式滑块控制。

## 🚀 功能特性
- ✅ 自定义函数输入
- ✅ 实时卷积过程可视化
- ✅ 滑块控制时间平移
- ✅ 四幅同步图表显示
- ✅ 实时计算信息展示

## 🌐 在线访问
如果已部署到GitHub Pages，可通过以下链接访问：
`https://[你的用户名].github.io/[仓库名]/`

## 📖 使用说明
1. 在左侧面板输入函数表达式
2. 点击"运行/更新卷积"按钮
3. 拖动滑块查看卷积过程
4. 观察右侧四幅图的实时变化

## 🔧 技术栈
- HTML5
- CSS3 (Flexbox布局)
- JavaScript
- Plotly.js (数据可视化)
